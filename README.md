# Write it! Kana

## DESCRIPTION

  Write it! Kana is an application that teaches the Japanese Hiragana and Katakana characters and handwriting. It is a hassle-free application. It provides features that will help you learn Japanese kana easier and faster. 

  One of its features is that it avoids you from getting stuck, wherein you have to go back to the reference on how to write a character. Also, this application provides write-on animation guide for practice, which makes learning extremely fast and stress-free. 

  Write it! Kana lets you view the Katakana and Hiragana characters to thoroughly understand their meaning without taking a look at books and websites. The following are more of the features of the application: 

-Visuals of the characters of the 2 kinds of Japanese handwriting. 
-Sketch area for writing the chosen character. 
- Practice button, write-on animation guide for characters.
-Test button, test to measure your knowledge in Katakana & Hiragana.  
-Audible pronunciation button for characters.
-Chart, to view the list of Hiragana and Katakana characters.
-User Progress, displays the userâ€™s performance status (progress and accomplished tests in the application).
